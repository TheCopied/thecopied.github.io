copymachine


A copymachine is composed of three parts:

  • The Code

    • A 179B bash file "copy.sh" endlessly copies the media within the folder "copyrighted material." The script renames the media to either 0 or 1, deletes the original, and repeats the process. Every time a file is copied $150,000 is added to the tally of accrued copyright fines.
    • The code can be seen here: 
      	#!/bin/bash
	cd copyrighted_material
	i=0
	while :
	do
		for a in *; do
			if [ $a = 0 ]; then
				b=1
			else
				b=0
			fi
			cp $a $b && rm $a
		((i++))
		done
		echo $((150000 * $i)) 
	done
  • The Media

    • Within the folder "copyrighted material" a 94kB mp4 video file named "0" depicts 20s from Frozen 2 (2019).
  • The Confession

    • A text file "confession.txt" provides an account of an unknown criminal confessing to the crimes committed by their copymachine.
